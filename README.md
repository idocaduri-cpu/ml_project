# ml_project

Nova Academy course-cancellation (`Dropped_Course`) prediction — Stage 1 EDA.

## Setup

```
git clone https://github.com/YotamEviatar-stu/ml_project.git
cd ml_project
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Open `train.ipynb` in VS Code and select the `.venv` interpreter as the kernel (top-right "Select Kernel").

See `stage1_plan.md` for the notebook structure and variable priority order.
